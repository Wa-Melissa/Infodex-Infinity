(inner) closeApp(forceopt)

Closes the desktop application, optionally forcing the closure. This function checks if the application exit is locked. If it is locked and the force parameter is not set to true, it displays a SweetAlert notification indicating that the app cannot be closed.
Parameters:
Name Type Attributes Default Description
force boolean <optional>
 false Indicates whether to force close the application. If set to true, the application will close regardless of the exit lock state.

(async, inner) loadApp(appName)

Loads a specified desktop application. This function sets the exit lock state to false and updates the iframe's source to load the specified application. It also ensures that the application container is displayed.
Parameters:
Name Type Description
appName string The name of the application to be loaded. The corresponding HTML file should be located at "app/{appName}/app.html".

(inner) reloadApp()

Reloads the desktop application currently displayed. This function temporarily sets the source to a blank page and then resets it to its original source, effectively reloading the application.

(inner) updateClock()

Updates the displayed clock information. This function calculates the current day and remaining hours in a session based on the total time passed. It updates the inner HTML of the clock element to show the current day and the remaining hours left in the current day.
