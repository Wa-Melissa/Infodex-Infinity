(inner) createDOMReferences(request) → {Object}

Creates DOM references from a request object. This function takes an object where each key corresponds to a reference name, and each value is a string indicating the selector (ID, class, or other) for the DOM reference. The references are then stored in a new object, which is frozen to prevent any modifications.
